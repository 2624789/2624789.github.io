## Instalación

    $ curl https://install.meteor.com | sh

Si ya lo tenemos instalado, lo podemos actualizar

    $ meteor update

Si corremos este comando dentro de la carpeta de una aplicación, actualiza  la versión de Meteor de esa aplicación en particular.

## Crear Aplicación

    $ meteor create tareas

Este comando crea una carpeta con los archivos base de una aplicación
meteor.

    client/main.js        # a JavaScript entry point loaded on the client
    client/main.html      # an HTML file that defines view templates
    client/main.css       # a CSS file to define your app's styles
    server/main.js        # a JavaScript entry point loaded on the server
    package.json          # a control file for installing NPM packages
    .meteor               # internal Meteor files
    .gitignore            # a control file for git

## Correr Aplicación

    $ cd tareas
    $ meteor

## Directorios Reactivos

Almacenan estados temporales reactivos en el cliente. Son como objetos JavaScript, pero con reactividad incorporada.
    
    // Reactive Data Store
    this.state = new ReactiveDict()

    // Template Instance
    instance.state.get(...)
    instance.state.set(...)
    
## Métodos

Son funciones que se corren en el servidor, pero son llamadas desde el cliente. Como se ejecutan en el servidor, se supone que son confiables e ignoran callbacks allow/deny
